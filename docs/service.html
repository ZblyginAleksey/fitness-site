АЭРОБНЫЕ ПРОГРАММЫ

АЭРОБИКА

Энергичный урок классической аэробики с элементами хореографии. Рекомендован для всех уровней подготовки.

STEP

Аэробика с использованием step-платформы. Освоение базовых шагов и простейших комбинаций в Нагрузка средней интенсивности, рекомендуется для начинающих.

CROSS-trening

Перекрестная аэробная тренировка на все группы мышц.

СИЛОВЫЕ ПРОГРАММЫ

СУПЕР НОГИ

Cиловой класс для тренировки мышц ног и ягодиц. Нагрузка высокой интенсивности.

СУПЕР ТЕЛО

Cиловой класс для тренировки всех крупных мышечных групп, нагрузка средней и высокой интенсивности.

UP-BODY

Силовой класс направленный на развитие мышц груди, спины, и брюшного пресса.

ПРОГРАММЫ MIND BODY

ПИЛАТЕС

Класс направлен на поддержание мышечного тонуса, улучшения мышечного баланса, контроля над телом и развитие координации движения. Рекомендован для всех уровней подготовки.

HOT YOGA

Йога в помещении с температурой 35 градусов, программа рассчитана как для новичков, так и для "продвинутого" уровня. Время занятия 1.5 часа. Обязательно иметь при себе большое полотенце.
